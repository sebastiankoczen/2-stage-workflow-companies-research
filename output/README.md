# Output Directory

This folder contains generated intelligence reports.

Files are automatically created here during each weekly run:
- `stage1_chart_YYYYMMDD_HHMMSS.html` — Stage 1 discovery chart
- `stage1_raw_YYYYMMDD_HHMMSS.json` — Raw AI responses (audit)
- `stage2_input.json` — Top 10 companies for Stage 2
- `stage2_report_YYYYMMDD_HHMMSS.html` — Deep scan report (emailed)

All files are committed back to this repo after each run.
